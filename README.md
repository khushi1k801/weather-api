# Weather Forecast App 🌤️

A simple weather forecast web app built with React and Vite.

## 🚀 Features
- Get current weather of any city
- Responsive design
- Powered by OpenWeather API

## 🛠️ Tech Stack
- React
- Vite
- CSS
